# Error frecuente: Credenciales incorrectas

## Código
ERR-AUTH-001

## Categoría
Autenticación

## Mensaje mostrado al usuario
"Usuario o contraseña incorrectos."

## Causas posibles
- Contraseña mal escrita.
- Usuario inexistente.
- Cuenta desactivada.
- Bloqueo por intentos fallidos.
- Problema con el servicio de autenticación.

## Solución recomendada
1. Verificar que el usuario y la contraseña sean correctos.
2. Comprobar si la cuenta está activa.
3. Revisar si la cuenta se encuentra bloqueada.
4. Solicitar al administrador el restablecimiento de la contraseña si el error continúa.

## Palabras clave
login, inicio de sesión, contraseña, usuario bloqueado, credenciales
